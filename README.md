This repository provides tutorial code in C++ for deep learning researchers to learn PyTorch.

## Getting Started
- Fork/Clone and Install
```
$ git clone https://github.com/prabhuomkar/pytorch-cpp.git
$ chmod +x scripts.sh
$ ./scripts.sh install
```
- Build
```
$ ./scripts.sh build
```
