**Python Tutorial**: [https://github.com/yunjey/pytorch-tutorial](https://github.com/yunjey/pytorch-tutorial)

## Getting Started
- Fork/Clone and Install
```
$ git clone https://github.com/prabhuomkar/pytorch-cpp.git
$ chmod +x scripts.sh
$ ./scripts.sh install
```
- Build
```
$ ./scripts.sh build
```
